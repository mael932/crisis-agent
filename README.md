\# Crisis Response Agent

\### Timothée vs. Opera/Ballet — PR Workflow Challenge Submission



A multi-agent pipeline that triages a fast-moving celebrity PR crisis, routes work to the right stakeholders, produces aligned communications, and updates the plan when pressure escalates.



\## What This Is



A 5-agent Python pipeline powered by the Claude API. Each agent has a single, scoped job. They run in sequence and hand structured JSON to the next agent. The final output is a single submission.md file a PR team can pick up and act on immediately.



\## Agent Architecture



1\. TriageAgent — severity score, meme/sponsor/awards risk, hold-comment decision, hard deadlines

2\. RouterAgent — stakeholder priority queue, approval mapping, sequencing rationale

3\. DraftingAgent — raw drafts: sponsor holding line, reporter reply, manager brief, talent note

4\. ReviewerAgent — issues flagged per draft, revised finals, status (APPROVED/REVISED/REJECTED)

5\. EscalationAgent — diffs Phase B vs Phase A, updates routing and drafts, lists what is still valid



\## How to Run



1\. Install dependencies: pip install anthropic

2\. Set your API key: set ANTHROPIC\_API\_KEY=your\_key\_here

3\. Run Phase A: python main.py --phase a --save

4\. Run Phase A + B: python main.py --phase b --save



\## File Structure



\- main.py — full 5-agent pipeline

\- requirements.txt — dependencies

\- packets/packet\_a.md — Phase A crisis input

\- packets/packet\_b.md — Phase B input

\- output/submission.md — generated Phase A and B outputs



\## Scoring Alignment



\- Crisis judgment and triage (35%) — TriageAgent scores severity with rationale, RouterAgent sequences by deadline and approval chain

\- Adaptation to new information (20%) — EscalationAgent diffs Phase B vs Phase A explicitly

\- Cross-stakeholder coordination (15%) — four separate stakeholder outputs with per-channel deadlines

\- Ease of handoff (15%) — single output file, clear agent labels, PR team can read and act immediately

\- Workflow credibility (15%) — JSON handoffs between agents are visible, reviewer catches violations



\## Human Edit Disclosure



All outputs are generated by the pipeline. A human operator should review all outward-facing copy before sending.
