The scenario

A 22-second clip of Timothée Chalamet goes viral. A luxury sponsor is escalating. A reporter wants a statement. The Oscars are hours away. Nothing has been confirmed or denied.

5:30 PM — Phase A

The clip breaks

Timothée is off-phone until 6:05. The sponsor needs a holding line by 5:50. A reporter is publishing by 6:30. The agent triages, routes, drafts, and reviews — all before talent is reachable.

6:15 PM — Phase B

Sponsor escalates

Maison Valeur threatens to pause all brand activity unless a credible plan arrives by 6:35. A second sponsor calls. Another reporter leaves a voicemail. The agent diffs the new situation and updates — without restarting.

Agent pipeline
01
Triage Agent
Scores crisis severity 1–10. Assesses meme risk, sponsor risk, awards-night risk. Sets hard deadlines. Decides whether to hold comment.
Phase A
02
Router Agent
Builds the stakeholder priority queue. Maps approval chains. Sequences the next 30–60 minutes of actions. Lists what NOT to do yet.
Phase A
03
Drafting Agent
Produces four communications: sponsor holding line (≤40 words), reporter reply, manager internal brief, and a note for talent at 6:05.
Phase A
04
Reviewer Agent
Critiques every draft. Flags voice violations, unverified claims, word-count breaches. Revises and approves finals. Shows raw → final diff.
Phase A
⏸
Human Review — Voicemail Triage
The pipeline pauses when audio signals are detected. The operator listens, provides transcripts, and the agent resumes with enriched context. The system knows what it cannot do alone.
Phase B
05
Escalation Agent
Diffs Phase B against Phase A. Updates routing and drafts. Produces the credible plan the sponsor needs by 6:35. Lists what's still valid. Does not restart from scratch.
Phase B
Run it

Three commands

01 pip install anthropic install
02 set ANTHROPIC_API_KEY=your_key authenticate
03 python main.py --phase b --save run full pipeline

Output saved to output/submission.md — Phase A triage, routing, drafts, and reviews, followed by Phase B escalation and updated plan. Drop in a new packet and the same logic runs on a completely new crisis.
